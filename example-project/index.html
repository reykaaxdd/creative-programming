Огляд проєкту

Цей проєкт — інтерактивна візуальна новела, яка знайомить користувача з історичними подіями епохи Наполеонівських війн. Гравець проходить через серію сцен, кожна з яких супроводжується ілюстраціями та короткими текстовими описами.

Мета проєкту:

Створити захопливий освітній інструмент, що поєднує історичний наратив із сучасними технологіями інтерактивності.

Основні завдання:

  • Розробка послідовних сцен із зображеннями та текстом
  • Інтуїтивна навігація між етапами історії
  • Використання JavaScript для керування логікою переходів

Технічна реалізація

Використані технології:

  • HTML5/CSS3 для структури та стилізації сторінки
  • JavaScript для інтерактивності та переходів між сценами
  • Використання зображень для створення атмосфери

Особливості реалізації:

  • Динамічна зміна фону відповідно до сцени
  • Кнопки для переходу між частинами історії
  • Адаптивний дизайн для різних пристроїв

Особливості проєкту

  • Історичний сюжет із візуальним супроводом
  • Простий та зрозумілий інтерфейс
  • Можливість легко повертатися до попередніх розділів

Керування

  • Кнопка "Далі" — перехід до наступної сцени
  • Кнопка "Back" — повернення до головного меню
